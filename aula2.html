Click Customizado

HTML

Código de click.html para formar a estrutura base e a div que receberá os cliques.

CSS

Código de click.css para formatar os estilos dos elementos e criar a animação de onda keyframes e formatar a div.

JAVASCRIPT

Código de click.js para capturar durante o clique do mouse a posição/coordenadas XY do mouse, e criar nessa coordenada um elemento span com a classe circulo que contém a animação de onda.
